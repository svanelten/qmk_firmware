# LordSVE: unusable german layout
Another try to get all the characters in a usable pattern.

Uses these features:
- Audio (Startup song, Goodbye song, music mode to annoy people+ qq	xyyk,)
- LED In-Switch
- LED breathing

## Keymaps
Layout available at the [keyboard-layout-editor.com](http://www.keyboard-layout-editor.com/#/gists/ee2b02a7d676db10a6476eddd8aee76f).
